# Play-Pause-Perform-Gaming-Habits-and-Student-Success

uniGame: Comprehensive Gaming & Academic Performance Analysis
Overview
This repository performs a comprehensive analysis of the "uniGame" dataset (Hugging Face: ismail31415/uniGame) to explore relationships between gaming behavior and academic performance. The notebook runs four research questions (RQ):

- RQ1 — Predict student CGPA using multiple regression models (Linear models, tree-based, boosting, SVR, KNN, etc.).
- RQ2 — Identify distinct student profiles via clustering (KMeans, GMM, Hierarchical, Spectral).
- RQ3 — Unsupervised behavioral structure analysis using dimensionality reduction (PCA, t-SNE, UMAP).
- RQ4 — Feature importance analysis (Random Forest, Gradient Boosting, XGBoost, permutation importance, SHAP).
